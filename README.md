React Task Manager

A simple task management application built with React.
It allows users to add, delete, complete, and filter tasks with data saved in localStorage.

FEATURES

Add new tasks
Delete tasks
Mark tasks as completed
Filter tasks (All / Active / Completed)
Task counter (Total / Done)
Clear all tasks
Data persistence using localStorage
Clean and responsive UI

TECH STACK

React (useState, useEffect)
JavaScript (ES6+)
HTML5
CSS3
Vite
LocalStorage API

INSTALLATION & SETUP

Clone repository:
git clone https://github.com/DavidTanyan/react-task-manager.git
Go to project folder:
cd react-task-manager
Install dependencies:
npm install
Run project:
npm run dev
Open in browser:
http://localhost:5173/
